## 基础概念

###String.inern()

```java
// 代码编译加载时,会在常量池中创建"abc",在运行时,会返回常量池的引用给str
String str1= "abc";
//代码编译时,会在常量池中创建"abc",在调用new时,会在堆中创建String对象,并引用常量池的字符串对象char[]数组.并返回String对象引用给str2
String str2= new String("abc");
//调用intern(),会去常量池中查找是否有等于该字符串的对象,如果有,则返回其引用对象,并赋值给str3
String str3= str2.intern();
assertSame(str1==str2);
assertSame(str2==str3);
assertSame(str1==str3)
```
